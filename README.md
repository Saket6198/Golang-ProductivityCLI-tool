## ‚ö° Quick Start  

### Prerequisites  
- Go 1.20+ ([Install Guide](https://go.dev/doc/install))  
- Terminal with 256-color support (real hackers don't use defaults).  

### Installation  

- Clone the Repository
```
git clone https://github.com/saket6198/Golang-ProductivityCLI-tool
cd taskcli
go build -o taskcli
```
